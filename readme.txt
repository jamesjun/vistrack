# Install
Download git.exe from [https://git-scm.com]

# Required toolbox
Image processing toolbox
(optional) Parallel processing toolbox 

# Usage
To open visual tracking GUI, run "vistrack gui" ("GUI" also works)
To update, run "vistrack update"
To display help, run "vistrack help"
